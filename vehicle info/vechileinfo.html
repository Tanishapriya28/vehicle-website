Vehicle Info App

Admin App

The Admin App is a powerful tool designed for administrators to manage vehicle information efficiently. Developed using Android Studio, Figma, and Firebase Console, this app provides a comprehensive interface for performing CRUD (Create, Read, Update, Delete) operations on the vehicle database. Administrators can easily add new vehicle entries, update existing records, and remove outdated information, ensuring that the database remains accurate and up-to-date. This app is essential for maintaining a robust vehicle information system.
